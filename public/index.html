Installation Guide

This system is built with Laravel Framework (Version 10.x)

Server Requirements:

  • PHP 8.1 or higher
  • BCMath PHP Extension
  • Ctype PHP Extension
  • Fileinfo PHP Extension
  • JSON PHP Extension
  • Mbstring PHP Extension
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension
  • MySQL 5.7+ or MariaDB 10.2+
  • Composer
  • Node.js & NPM

Installation Steps:


  • Step 1: Clone/Download the Project

    • Download or clone the project files to your local development environment or server.
    git clone [repository-url]
    Or extract the downloaded ZIP file to your web server directory.

  • Step 2: Install Dependencies

    • Navigate to the project directory and install PHP dependencies using Composer:
    composer install
    Install Node.js dependencies:
    npm install

  • Step 3: Environment Configuration

    • Copy the environment file and configure your settings:
    cp .env.example .env
    Generate application key:
    php artisan key:generate
    Update the .env file with your database credentials and other settings.

  • Step 4: Database Setup

    • Create a MySQL database and run migrations:
    php artisan migrate --seed
    Or import the provided SQL file:
    Import database/e-shop.sql into your database

  • Step 5: Storage Setup

    • Create symbolic link for storage:
    php artisan storage:link
    Set proper permissions for storage and bootstrap/cache directories:
    chmod -R 775 storage bootstrap/cache

  • Step 6: Build Assets

    • Compile frontend assets:
    npm run dev
    Or for production:
    npm run build

  • Step 7: Run the Application

    • Start the development server:
    php artisan serve
    Open your browser and navigate to http://localhost:8000

  • Step 8: Admin Access

    • Use the default admin credentials to access the admin panel:
    Email: admin@gmail.com
    Password: 1111
    Admin URL: your-domain.com/login
